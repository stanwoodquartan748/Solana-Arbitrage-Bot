## ⚙️ Basic settings explained

### Wallet
Your wallet holds the funds used for trades. Use a wallet you control. Keep your recovery phrase private.

### Trade size
This is the amount used for each trade. Start small.

### Slippage
This is the price range you will accept before the trade no longer makes sense. Lower values can reduce risk, but they can also cause more failed trades.

### Scan interval
This is how often the bot checks for price gaps. A shorter interval checks more often. A longer interval uses fewer system resources.

### Profit threshold
This is the smallest price gap the bot should consider. Set it high enough to cover fees.

## 🛠️ Common problems

### The app does not open
- Make sure you extracted the zip file
- Try running the app as admin
- Check that your antivirus did not block the file
- Download the latest release again

### The bot cannot connect to the network
- Check your internet connection
- Close VPN or proxy tools for a test
- Restart the app
- Try again after a few minutes

### The wallet does not connect
- Confirm the wallet is installed
- Make sure the wallet extension or app is unlocked
- Reconnect the wallet inside the bot
- Restart the browser if the wallet uses one

### The app feels slow
- Close other heavy apps
- Lower the scan rate
- Restart Windows
- Use a stronger PC if needed
